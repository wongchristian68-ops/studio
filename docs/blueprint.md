# **App Name**: RestoConnect

## Core Features:

- User Authentication: Secure sign-up/sign-in via email, Google, or phone number using Firebase Auth with client, restaurateur, and admin roles.
- Loyalty Program: QR code scanning for stamp/point validation; automatic generation of unique QR codes per restaurant.
- Referral Module: Automatic generation of unique referral codes for each user; automatic reward attribution based on configurable rules.
- Google Review Management: Dedicated page to encourage customers to leave Google reviews; notifications for restaurateurs regarding negative reviews; review statistics display.
- Restaurateur Dashboard: Real-time overview of clients, validated stamps, rewards, active referrals, and recent Google reviews; configuration options for loyalty rules, referral rules, and automatic messages.
- Customer Interface: Display of stamps/points, personal QR code, rewards, history, and a direct link to leave a Google review.

## Style Guidelines:

- Primary color: Saturated violet (#8000FF) for an innovative, energetic feel.
- Background color: Very light desaturated violet (#F0E5FF).
- Accent color: Blue-violet (#7300E6) for highlighting and calls to action.
- Body and headline font: 'Inter', a grotesque-style sans-serif with a modern, machined look that will feel right at home in a dashboard setting.
- Modern and clear icons representing different aspects of the app: loyalty points, QR codes, rewards, and restaurant reviews.
- Clean and intuitive layout with clear sections for loyalty programs, referral management, and Google review tracking, optimized for mobile responsiveness.
- Subtle transitions and animations to enhance user engagement when navigating between sections and interacting with elements like QR codes or reward claims.
